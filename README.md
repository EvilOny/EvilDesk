# EvilDesk

Проект был вдохновлён модифицированным **Spotify Car Thing** под названием **Desk Thing**.

В моём случае клиентское приложение поставлено на Raspberry Pi 3 model A+ с 5-дюймовым сенсорным экраном, а серверная часть развёрнута на ПК. 

На данный момент реализовано управление аудио в ОС Windows через WinRT. В дальнейшем планируется развить проект в подобие Stream Deck с возможностью управления какими-либо приложениями.

## Зависимости

Перед запуском необходимо установить зависимости:

```
pip install -r requirments.txt
```
